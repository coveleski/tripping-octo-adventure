Assassins

Assassins is played with a group of players that reside near each other. They are given targets and eliminate each other until there is only one player left.


 

To set up:

Create a game under the “Create” tab.

  • A game should have at least four players

  • A game can be played with as few as two players

  • There is so upper limit to players in a game


 

To play:

  • Once the game is created:

    • Players will join via the “Join” tab

  • Once all players have joined:

    • Have the game creator start the game on the game screen

    • Initial targets will now be assigned

      • Player will know who they are targeting

      • Player will not know who is targeting them

  • Once targets are assigned:

    • Review the specific rules set up by the game creator on the game's screen

      • Typical rules include hitting the target with a sock, tagging them, or whispering in their ear

      • Your specific game may have “safe zones” set up where targets cannot be eliminated

    • Track down your target and avoid being taken out!
